A set of ready-to-use formz inputs to make your life easier!

## Features

- Email
- Password
- Confirmed Password

## Installation

Add this to your package's `pubspec.yaml` file:

```yaml
formz_inputs: ...
```

**OR** run

```yaml
flutter pub add formz_inputs
```
